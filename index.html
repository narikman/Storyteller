the problem

parents are busier than ever - reading time has become a luxury. children spend hours on passive screens, missing the warmth of shared stories. we believe every child deserves a bedtime tale, even on the busiest days.

the solution

an ai companion that generates unique fairy tales in seconds, narrated by warm voices. parents can create personalized stories with chosen characters, settings, and morals - making every night magical.

the magic

ai-generated narratives adapt to each child's age and interests. currently 5 built-in voices - expanding to 20+ voices including parents, friendly monsters, and wizards in future updates.

magical functions
  • create your own fairy tale - choose heroes, settings, moral lessons. ai crafts a unique story in under 2 minutes.
  • rich tale catalog - 10+ original stories with new tales added monthly. classics reimagined for modern children.
  • voice narration - 5 built-in voices now - warm, friendly, adventurous. expanding to 20+ including parent voice cloning in future updates.
  • ai co-writer - parents describe what they want - ai builds a complete story structure with characters, plot, and gentle lessons.
  • bilingual mode - stories in english and native languages. helps children learn naturally through emotional connection.

technology stack - what we used
python 3.11 pyside6 (qt) openai api (gpt-4) elevenlabs tts sqlite git - github fastapi (backend) html - css - js

the core application is built with python and pyside6 for desktop experience. openai api generates unique story structures based on user input. tts engine provides natural voice narration. we're continuously improving the ai pipeline to make stories more engaging and voices more expressive.
